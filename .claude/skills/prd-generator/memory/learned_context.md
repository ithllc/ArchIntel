# PRD Generator - Learned Context

## Conventions
- PRDs default to `docs/coding_implementations/` output directory
- 7-section format: Introduction, Target Audience, Scope, Features, User Stories, Tech Requirements, Success Metrics
- Features numbered as 4.1, 4.2, etc.
- User stories use italic format: *As a [persona], I want [action] so that [benefit].*
